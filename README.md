# ipl-score-prediction-deep-learning
Deep learning regression model to predict IPL match scores using TensorFlow and Keras.


# IPL Score Prediction using Deep Learning

## Overview
This project explores the use of deep learning for regression-based prediction of IPL match scores.

The implementation is based on guided learning resources and focuses on understanding preprocessing, model building, and evaluation using TensorFlow and Keras.

## Workflow
- Data preprocessing (handling missing values, encoding categorical features)
- Feature scaling
- Neural network model training
- Model evaluation using MAE and MSE

## Tech Stack
- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Matplotlib

## Status
Initial implementation completed. Further improvements and model optimization planned.
